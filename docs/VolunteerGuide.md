# Volunteer Guide

## Overview
Volunteers manage the physical checkpoints (Registration, Meals, Mentoring Sessions, Hall Entry) using the Digital Event Operations system.

## Using the Scanner
- Access the scanner via `/dev/event-operations` (or the production equivalent URL).
- Select your **Active Checkpoint** (e.g., Lunch Day 1).
- Ensure your device has internet access to sync scans in real-time.
- If you lose internet, continue scanning! The **Offline Queue** will store scans locally and automatically sync when connectivity returns.

## Error Handling
- If a scan shows `ALREADY SCANNED`, the team has already claimed that checkpoint.
- If a scan shows `BLOCKED` or `EXPIRED`, refer the team to the Admin Help Desk.
