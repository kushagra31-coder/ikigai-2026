# Admin Guide

## Overview
As an Administrator, you have full control over the IKIGAI Event Operations, Mentors, Teams, and the Leaderboard.

## Daily Operations
- **Announcements**: Broadcast live alerts to all participants via `/workspace/admin/announcements`.
- **Leaderboard Control**: Freeze, re-calculate, or publish the leaderboard via `/workspace/admin/leaderboard`.
- **Digital Passes**: Use the Admin Scanner to instantly Block or Reactivate compromised team passes.

## Logs
- All destructive actions (e.g., deleting teams) are recorded in the **Audit Logs**. Monitor this via `/workspace/admin/logs`.
