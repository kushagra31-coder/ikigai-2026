# 1. Purpose

The Admin Dashboard is the command center of IKIGAI 2026.

Every system operation originates from this dashboard.

It allows administrators to manage

• Teams

• Mentors

• Tracks

• Sessions

• Website Content

• Announcements

• Leaderboards

• Sponsors

• Analytics

• Settings

Only authenticated administrators may access this dashboard.
